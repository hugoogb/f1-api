F1 API endpoints

All endpoints return a json object when talking about a single team/driver or a json array when there is more than one

Teams

Available :team_name

:team_name -> lowercase and "-" instead of " " (space)

  • red-bull-racing
  • ferrari
  • mercedes
  • alpine
  • aston-martin
  • mclaren
  • haas-f1-team
  • alphatauri
  • alfa-romeo
  • williams

/api/teams

Get all teams info

/api/teams?drivers=yes

Get all teams info plus drivers of every team

/api/teams/:team_name

Get single team :team_name info

/api/teams/:team_name?drivers=yes

Get single team :team_name info plus drivers of the team

Drivers

/api/drivers

Get all drivers info, returned as json array

/api/drivers/:team_name

Get drivers of a single team :team_name

/api/drivers/profile/:driver_name

Get profile and stats of a single driver :driver_name

:driver_name -> lowercase and "-" instead of " " (space)
